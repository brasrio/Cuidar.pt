═══════════════════════════════════════════════════════════════
    🔧 PROBLEMAS COMUNS
═══════════════════════════════════════════════════════════════

❌ "Cannot GET /..."
   → Servidor não está rodando
   → Execute: node server.js no diretório server

❌ "Erro ao conectar com servidor"
   → Servidor não está rodando
   → Verifique se apareceu "Servidor rodando na porta 3000"

❌ Vejo endereço "C:/Users/..." na barra
   → Você está abrindo arquivo local
   → SEMPRE use localhost:3000

❌ Login não funciona
   → Você está em arquivo local
   → Acesse http://localhost:3000/login.html

❌ Dashboard aparece vazio
   → Servidor não está rodando
   → OU você está em arquivo local
   → Sempre use localhost:3000
